Dataset

The dataset used in this project is SMS Spam Collection Dataset It includes 5,574 sms messages, tagged according being ham(legitimate) or spam. The SMS Spam Collection is a set of SMS tagged messages that have been collected for SMS Spam research.

Machine Learning Model

The prepared dataset was implement and tested on different machine learning model and finally this dataset performed well with Multinomial Naive Bayes.

Note: In some cases the model may not provide proper accuracy. Please check your message properly.
